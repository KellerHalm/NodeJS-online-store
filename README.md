# Данный проект представляет собой простой онлайн-магазин, реализованный на Node.js с использованием Express и PosgreSQL. Проект демонстрирует основные возможности Node.js и позволяет создавать, читать, обновлять и удалять товары.

```bash
# Клонируйте репозиторий
git clone https://github.com/KellerHalm/NodeJS-online-store.git

# Перейдите в папку проекта
cd NodeJS-online-store

# Установите зависимости
npm install

# Создайте .env файл 

# Запуск сервера
npm run dev

# Запуск клиента
npm start
```
